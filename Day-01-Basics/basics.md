## 🔹 Flowchart
A flowchart is a diagram that visually represents the step-by-step logic of a program using symbols and arrows.

### Common Symbols:
- Oval → Start/End
- Rectangle → Process
- Diamond → Decision
- Arrow → Flow direction

---

## 🔹 Pseudocode
Pseudocode is a simple, English-like way of writing program logic without following strict programming syntax.

---

## 🔹 Variables
A variable is a named container used to store data that can change during program execution.

---

## 🔹 Data Types
Data types define the type of data a variable can store (e.g., integer, float, character, string).

---

## 🔹 Operators
Operators are symbols used to perform operations on variables and values (e.g., +, -, *, /, ==, &&).
